Portfolio

This site is my favorite and most complete personal project. I built it using HTML, CSS, and JavaScript, with the goal of showcasing not only my technical skills as a developer but also my background in design and the arts. I originally started with a warm red-based color palette, but later shifted to a cleaner black-and-white foundation with subtle red highlights, aiming for a more professional yet personal feel. I'm especially proud of a few technical aspects: I integrated EmailJS to handle contact forms more smoothly than a basic mailto, and I restructured the site from multiple HTML pages into a single-page layout with smooth scroll-based navigation. I also developed an interactive carousel that links to a dedicated projects section, where each project is detailed in its own scrollable area. Everything is designed to provide a clear, accessible, and visually thoughtful experience.

Maps

Maps is an interactive web app where users can search for predefined cities or add their own addresses, categorizing each location with a color-coded marker based on the type—such as points of interest, historical, religious, or cultural sites. The app includes a legend explaining the colors, and a button to display earthquake data from an XML feed covering Spain. Built using JavaScript, HTML, CSS, OpenStreetMap, and the Leaflet API, this was my first project working with APIs, which was a rewarding learning experience. Developed in a team of three during four sessions of six hours each, we focused on creating a clean and intuitive interface, using a grayscale color palette that complements the map’s tones for a cohesive and user-friendly design.

CalcuRedes

CalcuRedes is a network calculator designed to compute key data from an IP address, such as subnet masks, wildcards, broadcast addresses, and subnet details. Although the project had some technical challenges and time constraints, it was a valuable learning experience. Built with HTML, CSS, and JavaScript, this tool helped me and my teammates deepen our understanding of networking concepts. The interface features a clean design with blue and white tones, complemented by pixel-style typography to give it a retro-tech feel.

Memorium

Memorium was my first project as an intern at IES San Andrés, where we developed a memory game over two weeks in a group setting. None of us had prior experience with JavaScript, so the project involved extensive research and following many tutorials to learn as we built. The game features a gothic and Souls-like visual style, with multiple boards and game modes to keep players engaged. A fun easter egg activates if the username entered is "snake" — instead of the memory game, the classic Snake game appears. This nod is inspired by our Metal Gear-themed card deck, which includes characters named Snake. Despite the challenges of tackling JavaScript for the first time, I’m proud of the progress we made and the technical skills I developed during this intense and rewarding project.
